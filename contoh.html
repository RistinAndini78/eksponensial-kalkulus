Contoh Soal

Soal 1

Diberikan 𝑓(𝑥) = 22𝑥−1, carilah nilai dari 𝑓(2) dan 𝑓(½) adalah...

Jawaban:
f(2) = 22(2)-1 = 24-1 = 23 = 8
𝑓(½) = 22(½)-1 = 21-1 = 20 = 1
Pembahasan: Substitusi nilai x ke dalam fungsi 𝑓(𝑥).

Soal 2

Lukislah grafik fungsi 𝑦 = 2𝑥 dengan 𝑥 ∈ R

Jawaban:
Titik-titik grafik:
x = -2, y = 2⁻² = ¼
x = -1, y = 2⁻¹ = ½
x = 0, y = 2⁰ = 1
x = 1, y = 2¹ = 2
x = 2, y = 2² = 4
Pembahasan: Menghitung tiap titik grafik.

Soal 3

Tentukan penyelesaian f(x) = x³ untuk x = 3

Jawaban:
f(3) = 3³ = 27
Pembahasan:
Substitusi x = 3 ke dalam fungsi 𝑓(𝑥) = x³.

Soal 4

Tentukan penyelesaian f(x) = x² + 1 jika x = 5

Jawaban:
f(5) = 5² + 1 = 25 + 1 = 26
Pembahasan:
Substitusi x = 5 ke dalam fungsi 𝑓(𝑥) = x² + 1.
