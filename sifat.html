Sifat-Sifat Fungsi Eksponensial

1. Selalu Positif

Untuk setiap nilai x (baik positif, negatif, atau nol), nilai f(x) selalu positif. Grafik fungsi eksponensial selalu berada di atas sumbu x.

2. Memotong Sumbu-y di Titik (0,1)

Ketika x = 0, maka f(0) = a0 = 1. Jadi, grafik fungsi eksponensial selalu memotong sumbu-y di titik (0,1).

3. Monoton

- Jika a > 1: Fungsi akan monoton naik, artinya nilai f(x) akan terus bertambah seiring bertambahnya nilai x.
- Jika 0 < a < 1: Fungsi akan monoton turun, artinya nilai f(x) akan terus berkurang seiring bertambahnya nilai x.

4. Tidak Memotong Sumbu-x

Grafik fungsi eksponensial tidak akan pernah memotong sumbu-x, karena tidak ada nilai x yang dapat membuat ax = 0.

5. Pertumbuhan atau Peluruhan Eksponensial

- Pertumbuhan eksponensial: Jika a > 1, fungsi menggambarkan pertumbuhan yang semakin cepat seiring berjalannya waktu.
- Peluruhan eksponensial: Jika 0 < a < 1, fungsi menggambarkan peluruhan yang semakin lambat seiring berjalannya waktu.

6. Sifat-sifat Aljabar

Fungsi eksponen mengikuti sifat-sifat aljabar eksponen pada umumnya, seperti:
am * an = am+n
(am)n = am*n
am / an = am-n
